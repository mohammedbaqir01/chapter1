# Chapter 1

A small HTML learning project from an early web development chapter. This repository is part of my practice work while strengthening core software development skills and preparing for more advanced data, machine learning, and MLOps projects.

## Project Description

This project contains a simple HTML-based exercise. It is useful as a record of foundational web development practice, including basic page structure, content organization, and GitHub repository workflow.

## Tech Stack

- HTML
- Git & GitHub

## Installation

Clone the repository:

```bash
git clone https://github.com/mohammedbaqir01/chapter1.git
cd chapter1
```

No package installation is required for a static HTML project.

## Usage

Open `index.html` in a browser:

```bash
start index.html
```

On macOS or Linux, open the file with your preferred browser.
